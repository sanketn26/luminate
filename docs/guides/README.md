**Audience:** New users wanting to try Luminate

**You'll learn:**
- How to start Luminate locally
- How to write your first metric
- How to query metrics
- How to visualize in Grafana

### Development Setup
**Audience:** Contributors and developers

Set up a complete development environment with hot reload, testing, and debugging.

**You'll learn:**

- How to build from source

**Audience:** Operators deploying Luminate

Complete reference of all configuration options for production deployments.

**You'll learn:**

- Rate limiting tuning

**Audience:** API users and integration developers
